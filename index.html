ESP8266 Project

WiFi board microcontroller.

This is one of the experiments that I did on the esp8266 board. Starting from the board with the esp01 series which has 8 pins, here I am trying to function the board as a communication bridge between Arduino and the internet. I functioned this esp01 board as an http client that made _GET requests to the server in accordance with the data sent using serial communication by Arduino.

Router Board Hacking

Using opensource firmware openWRT

The router that I use is the TPLINK router with the tl-mr3020 series. The firmware that I use is openWRT, this firmware is a linux operating system that is specific to embedded devices. As the Linux OS firmware is also very flexible in its use, and the package for this firmware is quite a lot. In this project I use this router as a database and web server and connect with Arduino Nano to display messages on the LCD, as well as to develop input output so that it can communicate with sensors or other devices.

Hansip V1.0

Home security system with push notification.

I inspired this project from security officers in Indonesia who we usually call SATPAM (satuan pengamanan or in english security units). This tool is designed to provide push notifications to users when sensors detect interference. This SATPAM feature uses an rfid card and pin password to open, activate and shut down the system and alarm. This device uses Arduino Nano, ESP8266, RFID sensor, 4x4 membrane keypad, LCD and several sensors.
